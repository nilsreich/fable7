# Unterrichtsmaterial für die Lehrkraft (BGY 11 Mathematik)

Dieser Ordner enthält das Planungsmaterial der Lehrkraft. Er liegt bewusst
**außerhalb** von `src/content/` und wird daher **nicht** auf der
Schüler-Website veröffentlicht.

| Datei | Inhalt |
| --- | --- |
| `01-paedagogisches-konzept.md` | Ausführliches pädagogisches Konzept mit lernwissenschaftlicher Begründung |
| `02-jahresplan.md` | Überarbeiteter, realistischer Jahresplan (42 Wochen, inkl. gebrochenrationaler Funktionen, Puffern und Kürzungskaskade) |
| `03-spickzettel-block1.md` | Stunden-Spickzettel Woche 1–10 (Basiswissen & Funktionen) |
| `03-spickzettel-block2.md` | Stunden-Spickzettel Woche 11–20 (Ganzrationale & gebrochenrationale F., Einstieg Differentialrechnung) |
| `03-spickzettel-block3.md` | Stunden-Spickzettel Woche 21–32 (Differentialrechnung, Kurvendiskussion, Scharen) |
| `03-spickzettel-block4.md` | Stunden-Spickzettel Woche 33–42 (Extremwertprobleme, Steckbriefe, Gauß) |
| `04-kritik.md` | Kritik des eingereichten Plans und Selbstkritik des Konzepts |

Grundlagen: Lehrplan Berufliches Gymnasium Mathematik Rheinland-Pfalz
(19.12.2014, Lernbereiche 1 und 2, Zeitrichtwert 160 h), schulischer
Arbeitsplan BGY 11 (Lernsituationen 1–7) sowie die Inhaltsseiten dieser
Website (`src/content/docs/`).
